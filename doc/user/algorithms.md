# Algorithms

The algorithm section represents the core of your evaluation workflow. This is the part that mostly is implemented a new derived module classes for your own algorithms. 
Still, a few predefined modules are provided

## Unity
This module is the identity operator. I.e., it plain and simply just passes the data through without modification.

## Gauss
A Gauss filter is applied to the image

## Median
A median filter is applied to the image.
